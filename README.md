# Movie Ticket Booking System

A Java-based movie ticket booking system with GUI support using Swing.

## Features
- Movie selection
- Date & showtime selection
- Seat booking system
- GUI seat layout
- Payment simulation
- File-based seat saving
- OOP concepts implementation

## Technologies Used
- Java
- Swing GUI
- File Handling
- OOP

## Concepts Used
- Inheritance
- Interface
- Abstract Class
- Composition
- Collections
